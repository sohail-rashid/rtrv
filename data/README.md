# Test Corpus

This directory contains sample data for testing the search engine.

## Files

- `test_corpus.txt` - Small test corpus for unit tests
- `wikipedia_sample.txt` - Sample Wikipedia articles for benchmarking

## Format

Each line represents a document with format:
```
document_id|document_content
```

Example:
```
1|The quick brown fox jumps over the lazy dog
2|A journey of a thousand miles begins with a single step
```
